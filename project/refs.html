Basic

Command Usage Description
Initialize Git init Intializes the current directory and all child directories to be used by git
Clone a Repository clone <repoURL> Clones an existing repository into the current directory
Add File add <fileName> Adds the file into the staging area for commiting
Add All Files add -A Adds all changed files into the staging area for committing
Commit Staged Files commit -m "<message> Commits all the files staged and attaches the message with that group of changes
Create a Branch branch <branchName> Creates a new Branch based off of the current active branch
Checkout to Branch checkout <branchName> Changes the current active branch to <branchName>
Merge a Branch merge <branchName> Merges <branchName> into the current active branch
Pull from Remote pull Pulls all remote changes into the local version and attempts to merge any out of sync changes
Push to Remote push Pushs all local changes to the remote repository

Advanced

Command Usage Description
Set Upstream push -u origin <branchName> Sets the <branchName> to push to the origin server and then pushs all changes up
Show Remotes remote -v Shows all the remotes registered to be pushed and pulled from
Add a new Remote remote add <remoteName> <remoteURL> Adds the new remote to the repository under the current name
Set Upstream for different remote push -u <remoteName> <remoteBranchName> Sets the current branch to push to <remoteName>'s <branchName>
View All Branches branch -vv Shows all the branches and their last commit message
View the Log log Shows the history of the current branch
Pretty Log Print log --graph --decorate Shows a prettier history of the current branch
